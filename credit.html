GitHub Pages

GitHub는 분산 버전 관리 툴인 깃을 사용하는 프로젝트를 지원하는 웹호스팅 서비스로 전 세계에서 가장 인기있는 오픈 소스 코드 저장소입니다. 이 웹사이트는 GitHub Pages 서비스에서 무료로 호스팅되고 있습니다.

Infrastructure Technologies

HTML5

본 웹사이트의 최신 웹 마크업언인 HTML5로 제작 되었습니다. 시멘틱 태그를 사용하여 의미론적 마크업을 하고자 노력하였으며 HTML5의 문법을 준수하였습니다.

Infrastructure Technologies

CSS3

CSS3의 text, fonts, color, background, border, transform, transition, animation 등의 모듈을 활용해 화려하고 역동적인 면모를 갖추기 위해 노력하였습니다. 크로스브라우징을 고려해 범용성이 낮은 CSS3의 모듈의 사용은 지양하였으며, 크롬, 사파리 데스크톱 브라우저와 익스플로러9 이상의 버전에서 정상 작동 되는것을 확인하였습니다.

Infrastructure Technologies

SCSS

웹사이트 제작에 CSS 전처리 언어인 SCSS를 이용해 CSS 파일을 컴파일하였습니다. 간결한 문법을 통해 코딩 시간을 단축했음은 물론이고 CSS를 변수화하거나 함수를 이용하는 등의 방법으로 복잡한 상황에 쉽게 대응할 수 있었습니다. 또 하나의 스타일시트에 분산된 SCSS의 소스 파일을 IMPORT하는 방식으로 작업했기에 보다 효율적인 소스 관리가 가능헀습니다.

Infrastructure Technologies

SVG

해상도가 높은 디바이스에서 최적의 그래픽 퍼포먼스 구현을 위해 SVG 아이콘 등을 제작 적용하였습니다. 아이콘 외 Blur Filter 효과 또한 SVG를 통해 구현하였습니다.

JavaScript Libraries

jQuery

jQuery를 이용해 보다 쉽게, 뛰어난 액션 퍼포먼스를 구현했습니다. 또 사용자 편의성을 향상시키기 위해 각종 인터페이스를 제작하는데에 사용하였습니다. 본 웹사이트의 동적인 인터페이스 및 작동은 jQuery를 기반으로 만들어졌습니다.
